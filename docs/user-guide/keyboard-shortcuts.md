# Keyboard Shortcuts

## General

| Shortcut | Action |
|----------|--------|
| **Ctrl+O** | Open SEM images |
| **Ctrl+Q** | Quit application |

## Calibration & Setup

| Shortcut | Action |
|----------|--------|
| **Ctrl+K** | Set scale bar calibration |
| **Ctrl+R** | Set scan area |

## Analysis

| Shortcut | Action |
|----------|--------|
| **F5** | Analyze all images |
| **Ctrl+F5** | Reanalyze current image only |

## Export

| Shortcut | Action |
|----------|--------|
| **Ctrl+E** | Export all images to Excel |
| **Ctrl+Shift+E** | Export current image to Excel |

## Image Navigation

| Action | Control |
|--------|---------|
| Zoom in/out | Scroll wheel |
| Pan | Alt + left-click drag |
| Pan (alternate) | Middle-click drag |

## Grain Editing

| Action | Control |
|--------|---------|
| Select grain | Left-click on grain (overlay view) |
| Delete selected grain | Delete key |

All navigation controls (zoom, pan) work in the main canvas, the calibration dialog, and the scan area dialog.
